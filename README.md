# Colorfly Studio

Colorfly Studio es una aplicación web interactiva para generar paletas de colores dinámicas en distintos formatos.  
Permite visualizar combinaciones de colores en HSL y RGBA, copiar códigos HEX fácilmente y bloquear colores individuales mientras se generan nuevas paletas.

## Características

- Generación automática de paletas de colores
- Modos de color HSL y RGBA
- Visualización de códigos HEX
- Copia rápida del código HEX al hacer click
- Bloqueo individual de colores
- Cambio dinámico entre 6, 8 o 9 colores
- Diseño responsive y moderno

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript Vanilla
- Git
- GitHub Pages

## ¿Cómo se usa?

- Selecciona el tamaño de la paleta: 6, 8 o 9 colores
- Luego, selecciona el formato: RGBA o HSL
- Para generar nuevas paletas haz clic en Generar paleta
- Puedes hacer clic en el candado (🔓) para bloquearlo (ese color se mantiene cuando regeneras una nueva paleta)

## Decisiones técnicas

- Se utilizó JavaScript Vanilla sin frameworks para comprender mejor la manipulación del DOM y la lógica de interacción.
- Los colores HSL permiten generar paletas armónicas mediante separación de tonos.
- Los colores RGBA fueron ajustados para evitar tonos demasiado oscuros y mejorar la experiencia visual.
- Se separó la estructura del proyecto en archivos HTML, CSS y JavaScript para mantener una organización más clara y escalable.
- Se utilizaron clases dinámicas y eventos para controlar bloqueos de colores y generación de nuevas paletas.
